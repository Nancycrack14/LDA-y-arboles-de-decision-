En esta actividad trabajaré con una base de datos de la Liga MX que abarca los años 2016-2023. La base contiene información detallada sobre los partidos disputados, incluyendo datos como los equipos que participaron, los goles anotados por cada uno de los equipos en los diferentes tiempos del partido, el árbitro asignado a cada encuentro, el estadio donde se jugó, la jornada y la temporada a la que perteneció el partido. Además, la base de datos registra el resultado final de cada encuentro, especificando si el equipo local o visitante fue el ganador, o si el partido terminó en empate.

A lo largo de esta actividad, utilizaré esta información para realizar un análisis exploratorio y aplicar técnicas estadísticas y de modelado. El objetivo será identificar patrones relevantes dentro de los datos y utilizar modelos predictivos para obtener conclusiones sobre los resultados de los partidos, además de evaluar el desempeño de estos modelos. Se trabajará con diferentes enfoques de modelado, buscando generar predicciones que puedan ser útiles para entender mejor los resultados de los partidos en la Liga MX.

from sklearn.tree import DecisionTreeClassifier, plot_tree
from sklearn.model_selection import StratifiedKFold, cross_val_score
import matplotlib.pyplot as plt
import numpy as np
from sklearn.metrics import accuracy_score, f1_score

# 1. Modelo inicial del árbol de decisión
tree = DecisionTreeClassifier(random_state=0)
tree.fit(X_train[variables_relevantes], y_train)

# Visualizar el árbol de decisión inicial
plt.figure(figsize=(15, 10))
plot_tree(tree, filled=True, feature_names=variables_relevantes, class_names=["No Victoria", "Victoria"], rounded=True)
plt.show()

# Realizar predicciones con el árbol de decisión inicial
yhat = tree.predict(X_test[variables_relevantes])

# Evaluar el árbol de decisión inicial
accuracy_initial = accuracy_score(y_test, yhat)
f1_initial = f1_score(y_test, yhat)
print(f"Accuracy inicial: {accuracy_initial}")
print(f"F1-score inicial: {f1_initial}")

# 2. Realizar validación cruzada para encontrar el mejor valor de alpha (poda)
alpha_values = np.linspace(0.001, 0.2, 250)
cv_scores = []

# Realizar la validación cruzada para cada valor de alpha
for alpha in alpha_values:
    tree_pruned = DecisionTreeClassifier(ccp_alpha=alpha, random_state=0)
    cv_score = np.mean(cross_val_score(tree_pruned, X_train[variables_relevantes], y_train, cv=StratifiedKFold(n_splits=5), scoring='f1'))
    cv_scores.append(cv_score)

# Encontrar el mejor valor de alpha
best_alpha = alpha_values[np.argmax(cv_scores)]
print(f"Mejor valor de alpha (α): {best_alpha}")

# 3. Árbol podado con el mejor alpha
pruned_tree = DecisionTreeClassifier(ccp_alpha=best_alpha, random_state=0)
pruned_tree.fit(X_train[variables_relevantes], y_train)

# Visualizar el árbol podado
plt.figure(figsize=(15, 10))
plot_tree(pruned_tree, filled=True, feature_names=variables_relevantes, class_names=["No Victoria", "Victoria"], rounded=True)
plt.show()

# Realizar predicciones con el árbol podado
yhat_pruned = pruned_tree.predict(X_test[variables_relevantes])

# Evaluar el árbol podado
accuracy_pruned = accuracy_score(y_test, yhat_pruned)
f1_pruned = f1_score(y_test, yhat_pruned)
print(f"Accuracy del árbol podado: {accuracy_pruned}")
print(f"F1-score del árbol podado: {f1_pruned}")
